Cygwin 及相关内容简索，【革奴】须知

码农，独裁者、革奴；Unix、Windows、Cygwin；GUI、GPL；Cmd、Bash、Shell、GCC。它们是当今信息时代的三足撑天兵阵，犹如已发掘的秦始皇时代兵马俑阵，只不过，它们如今真实地活跃着。Windows及其Cmd、Unix及其Shell、GNU及其Bash，它们是计算机领域的三个操作系统及其行命令解译套件。现实社会当中有一群人――码农，尤为突出是有一群争自由(Free)的码农！他们致力于唤醒被电脑软件界独裁者――微软――所控制的大众，力图协助社会“革除微软帝国的奴役"，帮助大家在 Windows 环境上安装享有自由权利____Free____的开源应用软件。这里，Free 的含义是软件提供者尊重软件用户选择的“自由”,也有“免费”使用的含义。

在帝国 Windows 里，有一帮大闹天宫的孙行者依托 Windows 环境搞了一些开源软件，主要分为 GUI 图形视窗类, 如 Firefox, 以及基于 GNU 计划的模拟 Unix 的行命令环境类，如 Cygwin。

GUI 的含义是图形用户界面――Graphical User Interface，又称图形用户接口。

GNU 是一个自由授权开源操作系统。GNU 翻译为中文读音“革奴” , 字符 G 要发音“革”。GNU 的内容软件完全以 GPL 方式发布。GPL 即“通用公共授权”，是 GNU General Public License 的缩写。

Cygwin 是模拟 GNU 操作环境而不是 Unix 操作环境的操作系统。Cygwin 是码农中革奴阵营的一套兵器，但只是一套短兵器。

Cygwin 操作系统是“革奴计划”，亦即“GNU 计划”的主要技术目标成果。名称 GNU 来自于“GNU's Not Unix!” 这句话的缩写。因为 GNU 的设计类似于 Unix，但它又不包含具著作权的 Unix 代码，所谓具著作权即“非自由权”。GNU 的创始人理查德・马修・斯托曼将 GNU 视为“达成社会共享目的技术方法”。Cygnus Solutions 公司(后来为 RedHat 所并购)以自由软件基金会的 GNU 工具为基础研发了一系列软件, 并将它们移植到了 Windows 上, 就成了Cygwin。迄今，Cygwin 一直都是免费开源的。Cygwin 初期只有行命令 bash(Bash，是 Unix 中 shell 的变种), 及编译器 GCC 等等开发工具可供使用。现在则连 Mac OS X 视窗系统都已被移植成功，未来X视窗系统下的应用软件也将逐渐移植过来。

Bash 名称来自 Bourne shell(sh) 的一个双关语(Bourne again / born again)：Bourne-Again SHell。Bash 是一个命令处理器，运行于文本视窗中，并能处理和执行用户直接输入的命令。Bash 还能从文件中读取命令，这样的文件被称为“脚本”。和 Unix 的其他 shell 一样，它支持文件名替换(通配符匹配)、管道、here 文档、命令替换、变量，以及条件判断和循环遍历的结构控制语句。包括关键字、语法在内的基本特性全部是从 sh 借鉴过来的。其他特性，例如历史命令，是从 csh 和 ksh 借鉴而来。总的来说，Bash 虽然是一个满足 POSIX 规范的shell，但又有了很多功能扩展。

GCC(GNU Compiler Collection，GNU 编译器套件)是基于 GNU 开发的编程语言编译器。GNU 编译器套件包括 C、C++、 Objective-C、 Fortran、Java、Ada 和 Go 语言前端，也包括了这些语言的库(如 libstdc++，libgcj 等)。起初，GCC 只是为 GNU 操作系统专门编写的一款编译器。

GCC 是以 GPL 许可证所发行的免费开源软件，也是 GNU 计划的关键部分。GCC 现已被大多数类似 Unix 操作系统(如Linux、BSD、Mac OS X 等)采纳为标准的编译器，甚至在微软的 Windows 上也可以使用 GCC。GCC 支持多种计算机体系结构芯片，如 x86、ARM、MIPS 等，并已被移植到其他多种硬件平台上。GCC 原名为 GNU C 语言编译器(GNU C Compiler)，只能处理 C 语言。但它很快得到了扩展，变得可处理 C++，后来又扩展为能够支持更多的编程语言，如 Fortran、Pascal、Objective -C、Java、Ada、Go 以及各类处理器架构上的汇编语言等，所以改名 GNU 编译器套件(GNU Compiler Collection)。

Cygwin 提供了一套 Windows 平台下 GNU 行命令环境的组合式工具，Bash 和 GCC 是其中的两个利器。
